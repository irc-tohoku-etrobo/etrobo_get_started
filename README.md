### コミットルール
コミットメッセージには、プレフィックスを付けましょう。

| プレフィックス | 意味             |
|---------|----------------|
| add     | 機能/コード/ファイルの追加 |
| update  | 新規ではない機能追加     |
| remove  | 機能/コード/ファイルの削除 |
| change  | 仕様変更による機能変更    |
| fix     | バグの修正          |
| test    | テストの追加やテストの修正  |
| chore   | 自動生成ファイルをコミット  |
